# FOSSEE Website

This Website is made using astro framework a web framework for content-driven websites.

## Run Locally

Clone the project

```bash
git clone https://github.com/Jangidyogesh12/FOSSEE.git
```

Go to the project directory

```bash
cd FOSSEE
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run start
```

or

```bash
npm run dev
```
